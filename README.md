# Raccoon 🦝

Cliente http, permite mandar peticones http, para probar endpoints de bakend o cualquier api que se quiera implementar

## v1

El cliente actualmente cumple con una version simple de un clientes, algunas de las funciones son las siguientes: 

- Peticiones POST, PUT, PATCH, GET
- Soporte para params
- Soporte para token
- Soporte para body
- Multiples tabs

### Plataformas

Actualmente la aplicacion esta disponible para macOs, Windows y Linux

### Comando para ejecutar el proyecto 

npm run tauri dev

# Raccoon 🦝
A lightweight HTTP client built with React + Tauri + Rust.

## Features
- Fast native HTTP requests
- Status, response time and size
- JSON viewer
- Minimal UI

## Tech stack
- React
- Rust
- Tauri
- Reqwest
