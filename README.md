# Raccoon 🦝
A lightweight HTTP client built with React + Tauri + Rust.

## Features
- Fast native HTTP requests
- Status, response time and size
- JSON viewer
- Minimal UI

## Tech stack
- React
- Rust
- Tauri
- Reqwest
